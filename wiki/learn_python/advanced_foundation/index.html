字典

这里主要提一下defaultdict(). 在说明collections.defaultdict()之前，我们首先要提一下setdefault()方法。

Python 字典 setdefault() 函数和get()方法类似, 如果键不存在于字典中，将会添加键并将值设为默认值。

因此, Python中通过Key访问字典，当Key不存在时，会引发‘KeyError’异常。为了避免这种情况的发生，可以使用collections类中的defaultdict()方法来为字典提供默认值。

使用 list 创建 defaultdict

使用list作第一个参数，可以很容易将键-值对序列转换为列表字典。collections.defaultdict(list)使用起来效果和我们上面提到的dict.setdefault()比较相似，这种方法会和dict.setdefault()等价，但是要更快.

使用 int 创建 defaultdict
